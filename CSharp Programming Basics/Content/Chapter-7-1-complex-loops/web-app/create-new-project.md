#### Създаване на нов C# проект

Във Visual Studio създаваме ново **ASP.NET MVC уеб приложение** с език C#. Добавяме нов проект от [**File**] &rarr; [**New**] &rarr; [**Project…**]. Задаваме смислено име, например “**Fruits-Web-Game**”:
